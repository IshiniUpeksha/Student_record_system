# Student Record Management System (C)

A robust, console-based application built in C that manages student data using custom Structures and persistent File Handling. This project demonstrates core programming concepts including data validation, linear search algorithms, and intelligent file I/O.

## Key Features
- **Add Student Records**: Capture Student ID, Name, and GPA.
- **Smart File Persistence**: Automatically saves records to `students.txt`.
- **Intelligent Headers**: The system detects if the storage file is new and prints table headers only once to maintain a clean data format.
- **Search Functionality**: Efficiently locate any student record by their unique ID using a linear search algorithm.
- **Tabular Display**: View all current session records in a formatted table directly in the terminal.

## Technical Concepts Applied
- **Structures (`struct`)**: Used to create a custom data type for grouping student attributes.
- **File I/O**: Implemented `fopen` with both `"r"` (read) and `"a"` (append) modes to manage data without overwriting.
- **Control Flow**: Utilized `do-while` loops for the menu interface and `if-else` blocks for logic branching.
- **Flag Logic**: Used boolean-style integer flags (`isNewFile`, `found`) to control program behavior.

## File Structure
- `student_record_system.c`: The primary source code containing the logic and menu.
- `students.txt`: The persistent database file generated by the program.
- `README.md`: Project documentation.
